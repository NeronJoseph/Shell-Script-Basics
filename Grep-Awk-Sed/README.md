# GREP

Global Regular Expression Print. This is a find command used to find patterns in text. 

# SED

This command is a find and replace command. 

## Regular expressions
Regular experssions are made up of:
1. Anchors - Specify the position
2. Character sets - What is searched 
3. Modifiers - Specify how many times the previous character set is repeated

### Anchors
1. ^ - at the begenning of a line 
2. $ - at the end of a line 

*Note: If ^ is not placed at the begenning of $ is placed at the end, the two won't act as anchors*
